**Từ điển tiếng Trung tĩnh, nhanh, mạnh mẽ • Hơn 125.000 từ**

Centi là từ điển Trung-Việt hiện đại, chạy hoàn toàn trên GitHub Pages (không cần backend), được thiết kế dành cho người học tiếng Trung nghiêm túc và thích dùng từ phong phú.

### ✨ Tính năng nổi bật

- **125.000+** từ vựng từ CC-CEDICT (cập nhật thường xuyên)
- Tìm kiếm nhanh theo **Hanzi** (giản thể & phồn thể), **Pinyin**, **Nghĩa tiếng Việt**
- **Animation nét bút** (Stroke Order) đẹp mắt
- Giao diện sạch, thân thiện, tối ưu trên mobile
- Hoàn toàn tĩnh, load nhanh, có thể dùng offline (sau khi cache)
- Không quảng cáo, không tracker

### 🛠 Công nghệ sử dụng

- **Frontend**: React + Vite
- **Tìm kiếm**: Fuse.js (fuzzy search)
- **Stroke Order**: Hanzi Writer
- **Dữ liệu**: CC-CEDICT + Make Me a Hanzi
- **Deploy**: GitHub Pages
